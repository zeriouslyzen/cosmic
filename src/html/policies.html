Return Policy

Last Updated: December 2024

Return Window

Items may be returned within 30 days of delivery for a full refund, provided they are in original condition with tags attached.

Return Process

  1. Contact us at returns@cosmicdeals.com with your order number
  2. We will provide return instructions and a return authorization
  3. Ship the item back using the provided label
  4. Once received and inspected, we will process your refund

Refund Processing

Refunds will be processed to the original payment method within 5-10 business days after we receive the returned item.

  • Custom or personalized items

We currently do not offer direct exchanges. Please return the item and place a new order for the desired product.
